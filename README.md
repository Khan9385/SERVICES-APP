# Seva Pakistan

A static frontend prototype for a Pakistan-focused services marketplace.

## Features

- Painter, plumber, electrician, carpenter, house cleaner, car cleaner, and mechanic categories
- Provider cards with reviews, prices, response times, and specialties
- Search, category filtering, and sorting
- Profile modal with testimonial previews
- Hire request modal

## Run

Open `index.html` in a browser.
